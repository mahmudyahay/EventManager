# Event Manager

A Django-based web application for managing events, users, and administrators.  
This project provides user registration, login, task/event management, and email notifications.

---

## 🚀 Features

- **User Management**
  - User registration with profile image, email, phone, role, and password.
  - Login and logout functionality with session management.
  - Password reset via email with autogenerated secure codes.

- **Admin Management**
  - Separate admin login with role validation.
  - Redirected dashboards for admins and users.

- **Task/Event Management**
  - Create, view, and manage tasks/events.
  - Tasks linked to users via `ForeignKey`.
  - Task status tracking: **Upcoming**, **Canceled**, **Completed**.

- **Email Notifications**
  - Password recovery and other email notifications via Gmail SMTP.

---

## 🛠 Tech Stack

- **Backend**: Django 5.1
- **Database**: PostgreSQL
- **Frontend**: Django Templates (HTML, CSS, Bootstrap)
- **Email Service**: Gmail SMTP
- **Other Tools**: Python 3.12+, Pillow (for image upload)

## 📂 Email Service setup

EMAIL_PORT = 587
